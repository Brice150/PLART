## Run Locally

Clone the project

```bash
  git clone https://github.com/Brice150/3D-Printing.git
```

Go to backend directory 
--> Run application on Intellij

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve -o
```
