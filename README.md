# A Music Application

This application is a FullStack demo of my skills in Angular and Spring Boot.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Brice150/Music-And-Me.git
```

Go to backend directory 
--> Run application on Intellij

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve -o
```
